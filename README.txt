# Assignment 0

Given user input, this program will print a downward block diagonal of size n.
Assume that the given user input is a positive integer, and is less than 999.
If the input to the program was 4, the output would look like this:

+-+
| |
+-+-+
  | |
  +-+-+
    | |
    +-+-+
      | |
      +-+
